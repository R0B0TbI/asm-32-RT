# 🛠 ASM-32-RT: x86 32-bit Assembly Development

Учебный проект для разработки и тестирования программ на Ассемблере (NASM) в гибридной среде.

## 🏗 Архитектура окружения

Проект поддерживает два режима работы:
1.  **Локально (MacBook M1):** Через Docker-контейнер с эмуляцией `linux/386` (QEMU).
2.  **Удаленно (VDSina Ubuntu):** На реальном x86_64 железе с поддержкой 32-битных библиотек.
